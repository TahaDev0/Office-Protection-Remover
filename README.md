<div id="important-notes"></div>

### ⚠️ ملاحظات مهمة

*   **كلمة مرور الفتح (Open Password)**: هذا البرنامج **لا يستطيع** استعادة أو إزالة كلمة المرور التي تطلبها عند **فتح** الملف. هو متخصص في إزالة الحماية الداخلية (حماية التعديل، حماية الصفحات، حماية VBA، ...).
*   **الملفات المشفرة بالكامل**: إذا كان الملف مشفراً بالكامل (Encrypted)، قد لا يمكن فتحه بعد المعالجة.
*   **الملفات التنفيذية**: ACCDE, MDE هي ملفات تنفيذية وقد تكون إزالة الحماية محدودة بها (مثلاً، قد لا يمكن استعادة أكواد VBA المصدرية).
*   **النسخ الاحتياطي**: مهم جداً أخذ نسخة احتياطية من ملفاتك قبل البدء بعملية المعالجة.
*   **التوافق**: إصدارات Office المختلفة قد تتعامل مع الحماية وطرق إزالتها بطرق مختلفة، مما قد يؤثر على فعالية الأداة في بعض الحالات النادرة.

<div id="Eimportant-notes"></div>

### ⚠️ Important Notes

*   **Open Password**: This tool **cannot** recover or remove the password required to **open** the file. It specializes in removing internal protections (edit protection, sheet protection, VBA protection, ...).
*   **Fully Encrypted Files**: If the file is fully encrypted, it might not be possible to open it after processing.
*   **Executable Files**: ACCDE, MDE are executable files, and protection removal might be limited (e.g., original VBA source code might not be restorable).
*   **Backup**: It is crucial to create a backup of your files before starting the processing.
*   **Compatibility**: Different Office versions may handle protection and removal methods differently, which might affect the tool's effectiveness in rare cases.
